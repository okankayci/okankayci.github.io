

# Okan Kaycı - Kişisel Web Sitesi

Bu proje, [okankayci.github.io](https://okankayci.github.io) adresinde yayınlanan kişisel portföy sitesidir. GitHub Pages kullanılarak barındırılmaktadır.

## Amaç

Bu site, Okan Kaycı'nın:
- Özgeçmişini,
- Projelerini,
- Blog yazılarını,
- İletişim bilgilerini

tanıtmak amacıyla hazırlanmıştır.

## Özellikler

- Mobil uyumlu modern tasarım
- Proje ve deneyim bölümleri
- Blog veya makale paylaşım alanı (isteğe bağlı)
- İletişim formu veya sosyal medya bağlantıları

## Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript
- GitHub Pages
