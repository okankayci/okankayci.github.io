

# Vardiya Takip Uygulaması

Bu Flutter tabanlı mobil uygulama, kullanıcıların vardiya programlarını kolayca takip etmelerini sağlar. Kullanıcılar çalıştıkları vardiyaları manuel olarak takvim üzerinde işaretleyebilir ve günlük görevlerini düzenleyebilir.

## Özellikler

- Aylık takvim görünümü ile vardiya planı oluşturma
- Esnek vardiya tipi desteği
- Otomatik maaş hesaplama 
- İstatistikler
- Günlük görev sayfası
- Kullanıcı dostu, sade arayüz
